# Role UI Map

## Super Admin

Full access to dashboard, users, roles, settings, customers, RFQs, campaigns, inventory, and reports.

## Admin

Access to dashboard and all operational modules.

## Sales User

Focuses on customers, RFQ creation, RFQ pipeline, quote tracking, and deal status.

## Marketing User

Focuses on campaigns, audience selection, and campaign metrics.

## Inventory Manager

Focuses on products, stock levels, and RFQ inventory reservations.
